#### 素材 A：Meta 發布 Muse Spark — 開源之王轉身做閉源（4/8–9）
- **事件日期：** 2026-04-08 至 2026-04-09
- **來源：** [Meta 官方](https://about.fb.com/news/2026/04/introducing-muse-spark-meta-superintelligence-labs/)、[CNBC](https://www.cnbc.com/2026/04/08/meta-debuts-first-major-ai-model-since-14-billion-deal-to-bring-in-alexandr-wang.html)、[Fortune](https://fortune.com/2026/04/08/meta-unveils-muse-spark-mark-zuckerberg-ai-push/)、[gHacks](https://www.ghacks.net/2026/04/09/meta-launches-muse-spark-its-first-proprietary-ai-model-with-no-open-source/)、[The Register](https://www.theregister.com/2026/04/08/meta_muse_spark/)
- **摘要：**
  - Meta Superintelligence Labs（由前 Scale AI 聯合創辦人 Alexandr Wang 領軍）發布首款模型 Muse Spark
  - 代號 Avocado，原生多模態推理模型，支援語音/文字/圖像輸入，文字輸出
  - **重點：這是 Meta 第一個閉源模型**，不提供模型權重下載，僅限 meta.ai 和邀請制 API 使用
  - 與 Zuckerberg 2024 年發表的「Open Source AI is the Path Forward」公開信立場完全矛盾
  - 將於數週內部署到 Facebook、Instagram、WhatsApp、Messenger 和 Ray-Ban Meta AI 眼鏡
  - 社群反應兩極：開源社群批評「背叛」，分析師則認為這是商業化必然
- **推薦理由：** 本週最大的產業敘事轉折之一。Meta 從開源旗手轉向閉源，與週一 Gemma 4 開源的報導形成強烈對比。極適合作為 Breaking News 或回顧中的重要補充。

#### 素材 B：Anthropic 發布 Claude Managed Agents — 雲端 Agent 即服務（4/8）
- **事件日期：** 2026-04-08
- **來源：** [SiliconANGLE](https://siliconangle.com/2026/04/08/anthropic-launches-claude-managed-agents-speed-ai-agent-development/)、[Help Net Security](https://www.helpnetsecurity.com/2026/04/09/claude-managed-agents-bring-execution-and-control-to-ai-agent-workflows/)、[The New Stack](https://thenewstack.io/with-claude-managed-agents-anthropic-wants-to-run-your-ai-agents-for-you/)、[9to5Mac](https://9to5mac.com/2026/04/09/anthropic-scales-up-with-enterprise-features-for-claude-cowork-and-managed-agents/)
- **摘要：**
  - Anthropic 推出 Claude Managed Agents（公開 Beta），一套可組合 API 用於建構與部署雲端託管 Agent
  - 提供沙箱執行、檢查點、憑證管理、範圍權限、端到端追蹤等功能
  - 定價 $0.08/session hour + 標準 token 費用
  - 早期採用者：Notion、Asana、Rakuten、Sentry、Vibecode
  - 可透過 Console、Claude Code 和新 CLI 建構
- **推薦理由：** 與週三 MCP Agent 安全治理主題形成「從框架到平台」的延伸敘事。Anthropic 不只在定義安全標準，也在搶佔 Agent-as-a-Service 平台位置。
