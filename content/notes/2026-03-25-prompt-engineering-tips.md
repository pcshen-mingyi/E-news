## 為什麼 Prompt 很重要？

同樣的 AI 模型，不同的 Prompt 可以產出天差地別的結果。好的 Prompt 就像好的問題，能引導 AI 給出更精準的回答。

## 實用技巧

### 1. 明確角色設定

告訴 AI 它應該扮演什麼角色：

> 你是一位資深的 Python 開發工程師，請幫我 review 以下程式碼...

### 2. 提供具體範例

用 Few-shot 方式給 AI 看你想要的輸出格式：

> 輸入：蘋果 → 輸出：水果
> 輸入：汽車 → 輸出：交通工具
> 輸入：鋼琴 → 輸出：？

### 3. 分步驟思考

要求 AI 一步步推理，而不是直接給答案：

> 請一步步思考這個問題，先分析原因，再提出解決方案。

### 4. 限制輸出格式

明確指定你要的格式：

> 請用 JSON 格式回覆，包含 title、summary、tags 三個欄位。

## 常見錯誤

- 指令太模糊：「幫我寫個東西」 → 「幫我寫一封商務合作邀請信，對象是 XX 公司」
- 沒有提供上下文：AI 不知道你的背景
- 期待一次完美：多輪對話逐步優化效果更好
